# [SCRIPT Specification](./readme.md)

## Requirements

Script devices shall be required to:

1. reduce RAM;
2. reduce ROM;
3. reduce complexity;
4. be portable;
5. maximize code reuse;
6. minimize code surface area;
7. have international language support;
8. Operate in a safe manner;
